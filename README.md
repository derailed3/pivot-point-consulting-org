# Pivot Point Consulting -- Organizational Knowledge Base

[PLACEHOLDER: Company tagline]

This repository is the single source of truth for Pivot Point Consulting's organizational documentation, policies, processes, templates, and institutional knowledge.

## Quick Start

1. Clone the repository
2. Browse directories by functional area (see table below)
3. Use Claude Code skills (`/new-policy`, `/new-proposal`, etc.) to generate content
4. Submit changes via pull request

### Local Documentation Site

```bash
pip install -r requirements.txt
mkdocs serve
```

Opens at `http://127.0.0.1:8000` with live reload.

| Directory | Purpose |
|-----------|---------|
| `docs/` | Documentation (tutorials, how-to guides, reference, explanation) |
| `policies/` | HR, security, compliance, data governance, operations policies |
| `marketing/` | Brand standards, case studies, blog posts, social media |
| `client-delivery/` | Proposals, SOWs, presentations, reports |

| `business-development/` | Partner programs, alliances, GTM strategies |

| `knowledge/` | FAQ, knowledge articles, decision logs |

| `internal-comms/` | Newsletters, announcements, town halls |
| `project-management/` | Charters, risk registers, status reports, lessons learned |
| `competitive-intelligence/` | Competitor profiles, win/loss analysis |
