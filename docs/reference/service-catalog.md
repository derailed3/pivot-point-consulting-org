# Service Catalog

## Overview

This document catalogs the services offered by Pivot Point Consulting, organized by delivery model.
Each service includes a description, typical engagement parameters, and key deliverables.
Use this catalog as a reference when scoping new engagements, preparing proposals, and positioning PPC capabilities with prospective clients.

## Delivery Models

PPC offers three primary delivery models, each suited to different client needs and engagement types.

| Model | Description | Billing Basis |
|-------|-------------|--------------|
| **Staff Augmentation** | PPC consultants embedded within the client's team, working under client direction | Hourly or weekly rates |
| **Project-Based** | Defined scope with specific deliverables, milestones, and timelines managed by PPC | Fixed fee, capped T&M, or milestone-based |
| **Advisory** | Expert guidance, assessments, and strategic recommendations without hands-on implementation | Retainer, phase-based, or fixed fee |
